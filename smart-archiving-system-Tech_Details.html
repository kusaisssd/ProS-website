الوثائق والأرشفة

نظام شامل لإدارة جميع أنواع الوثائق

▼
يدعم جميع أنواع الملفات: PDF, Word, Excel, صور, فيديو, ملفات هندسية (DWG, PSD) وأي نوع آخر بشكل مرن. يتضمن عنوان وتفاصيل المستند، المشروع والقسم، التصنيفات، السمات والكلمات المفتاحية، ودرجة السرية والتشفير. جميع التعريفات ديناميكية قابلة للتعديل حسب العمل.

الوظائف الأساسية

  • رفع ملفات متعدد (Drag & Drop)
  • دعم حجوم حتى 500MB
  • معاينة مباشرة (PDF, Images)
  • تنزيل وطباعة
  • فهرسة الوثائق
  • البحث والفلترة وفق بيانات المستندات
  • البحث داخل المستندات
  • ربط بالمراسلات والمشاريع
  • بنية شجرية مرنة متعددة المستويات

الوظائف الإضافية (اختيارية)

  • OCR — استخراج نص من صور وتحويل PDF
  • تحليل ذكي — تصنيف تلقائي واستخلاص البيانات
  • إصدارات الوثائق (Versioning)
  • سلة المحذوفات
  • سير عمل الموافقات وتواريخ انتهاء
  • أرشفة طويلة الأمد
  • أرشفة سحابية (Google Drive, OneDrive, Mega, Dropbox)

إدارة المراسلات

إدارة شاملة للمراسلات الواردة والصادرة والداخلية

▼
نظام شامل لإدارة جميع أنواع المراسلات مع تتبع كامل للحركة والردود. يتضمن رقم المراسلة (توليد تلقائي)، التاريخ، النوع، من/إلى، الموضوع، الإجراء المطلوب، والمرفقات.

الوظائف الأساسية

  • توليد تلقائي لأرقام المراسلات
  • إدارة الواردة والصادرة والداخلية
  • ربط المراسلات (رد على مراسلة)
  • عرض هرمي للمراسلات والردود
  • Timeline زمني تفاعلي
  • نظام مرفقات (رفع متعدد)
  • معاينة الملفات
  • Soft Delete (حذف آمن)
  • تتبع التعديلات (Audit Trail)
  • بحث أساسي وفلترة

الوظائف الإضافية

  • OCR و AI — استخراج نص وتحليل وتصنيف ذكي
  • التوقيع الإلكتروني وسلسلة موافقات
  • أرشفة تلقائية حسب الفترة الزمنية
📂

المتطلبات التقنية

البيئة التقنية

🖥️ متطلبات السيرفر

  • نظام التشغيل: Windows Server 2019+ أو Linux (Ubuntu 20.04+)
  • المعالج: 4 Cores أو أعلى
  • الذاكرة: 8GB RAM (16GB موصى به)
  • التخزين: 100GB (حسب حجم البيانات)
  • قاعدة البيانات: SQL Server 2019+ أو PostgreSQL 13+

🌐 متطلبات العميل

  • المتصفح: Chrome, Firefox, Edge, Safari (أحدث إصدار)
  • الاتصال: 1 Mbps أو أعلى
  • دقة الشاشة: 1366x768 أو أعلى
  • JavaScript: مفعّل
